Class (The Idea)
   │
   └── Instance (The "Living" example of that idea on your PC)
         │
         └── Object (The "Instance" when it is handled by a user or script)

ROOT (The Top Level)
└── Namespaces (Containers/Folders)
    ├── Root\CIMV2 (Most common: Hardware & OS)
    │   ├── Classes (Blueprints/Templates)
    │   │   ├── Win32_Process
    │   │   │   ├── Instance 1: chrome.exe (PID: 1234)
    │   │   │   ├── Instance 2: powershell.exe (PID: 5678)
    │   │   │   └── Instance 3: svchost.exe (PID: 910)
    │   │   ├── Win32_Service
    │   │   │   ├── Instance: WinDefend (Running)
    │   │   │   └── Instance: Spooler (Stopped)
    │   │   └── Win32_LogicalDisk
    │   │       ├── Instance: C: (NTFS, 500GB)
    │   │       └── Instance: D: (FAT32, 16GB)
    │   └── ... (thousands of other classes)
    │
    ├── Root\Default (Registry & system events)
    ├── Root\SecurityCenter2 (Antivirus & Firewall status)
    └── Root\WMI (Low-level hardware driver data)
